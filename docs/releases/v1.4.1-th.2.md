# crosspointTH v1.4.1-th.2

Thai maintenance release by **JTIAPBN.Ai**.

> **Pre-release / รุ่นทดสอบ:** This build passes automated tests, the upstream hardware safety gate,
> production compilation, and image-size checks. It remains a pre-release until the corrected Thai
> status title, tone-mark height, spacing, and Markdown indexing are confirmed on a physical device.

## Changes from v1.4.1-th.1

- Thai filenames/book titles are included in the 8-point status-bar font.
- High C90 tone variants are lowered proportionally to the active font size.
- Thai justification is limited to a subtle one-pixel boundary adjustment.
- Markdown inline parsing is span-based instead of appending one UTF-8 byte at a time, and long
  paragraph wrapping yields to the device scheduler.
- Lightweight local EPUB reading statistics are available from the reader menu.
- README is Thai-only and text-only; the CrossInk review is documented without importing its large
  history/sync/chart subsystem.

## Installation safety

Only the app image `crosspointTH-firmware.bin` is distributed. No bootloader or partition-table
binary is included. Rename it to `firmware.bin` and use the existing CrossPoint SD-card update flow.

The hardware-sensitive safety gate confirms that partitions, platform code, HAL, board/power/display
drivers, OTA boot switching, and firmware flashing code match the recorded upstream base. The tested
device also successfully returned from `v1.4.1-th.1` to official CrossPoint firmware.

To roll back, download an official `firmware.bin` from
[CrossPoint Reader Releases](https://github.com/crosspoint-reader/crosspoint-reader/releases) and use
the same SD-card update flow.
