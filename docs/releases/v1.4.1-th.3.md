# crosspointTH v1.4.1-th.3

Thai maintenance release by **JTIAPBN.Ai**.

> **Pre-release / รุ่นทดสอบ:** This build passes automated tests, the upstream hardware safety
> gate, production compilation, and ESP32 image validation. Markdown table rendering still requires
> confirmation on a physical X3/X4 before this release is promoted to stable.

## Changes from v1.4.1-th.2

- Added GitHub-style Markdown pipe-table parsing for `.md` books.
- Rendered each table row as stacked, labeled fields so it remains readable on the narrow display.
- Kept table rows together across page breaks when space allows.
- Preserved escaped pipes and pipes inside inline code.
- Added parser and pagination regression coverage for Markdown tables.
- Replaced EPUB `Tab Row …, Cell …` labels with real `<th>` column headings, or `Column N` when
  semantic headings are absent.
- Advanced the EPUB section-cache version so previously opened books rebuild with the new labels.
- Updated the boot-screen edition label to `1.4.1-th.3`.

## Installation safety

Only the app image `crosspointTH-firmware.bin` is distributed. No bootloader or partition-table
binary is included. Rename it to `firmware.bin` and use the existing CrossPoint SD-card update flow.

The hardware-sensitive safety gate confirms that partitions, platform code, HAL, board/power/display
drivers, OTA boot switching, and firmware flashing code match the recorded upstream base.

To roll back, download an official `firmware.bin` from
[CrossPoint Reader Releases](https://github.com/crosspoint-reader/crosspoint-reader/releases) and use
the same SD-card update flow.
