## Highlights / สิ่งที่เพิ่ม

- Built-in Thai glyph coverage and automatic fallback for SD fonts without Thai glyphs.
- Thai dictionary word breaking for EPUB, TXT, and Markdown.
- Corrected stacked upper-vowel/tone-mark rendering, including `อึ่`, `อื้อ`, `ปึ้`, and `อ่ำ`.
- Faster Thai TXT/Markdown indexing by reusing segmentation results.
- Markdown headings, bold, italic, inline code, lists, block quotes, and link text.
- Thai-aware justification plus the upstream line-spacing/alignment settings.
- Thai UI translation and `crosspointTH` / `JTIAPBN.Ai` edition branding.

## Installation safety

- Supported targets: Xteink X3 and X4 only.
- The attached `.bin` is an ESP32-C3 DIO **app image**, not a complete-flash image.
- Preferred method: rename it to `firmware.bin` and use the existing CrossPoint SD-card firmware
  updater, or select it as a custom firmware in the upstream web flasher.
- If using esptool manually, the upstream app0 offset is `0x10000`. Do not write it at `0x0` or
  `0x8000`.
- Never disconnect power or reset the device while flash writing is in progress.
- Keep a downloaded copy of an official CrossPoint Reader firmware before installing this build.

No bootloader or partition-table binaries are attached to this release.

## Rollback

Download an official `firmware.bin` from
[CrossPoint Reader Releases](https://github.com/crosspoint-reader/crosspoint-reader/releases) and
install it using the same SD updater or USB custom-firmware route. The Thai build does not change the
upstream partition table, bootloader, HAL, power manager, display driver, or firmware flashing code.
Book layout cache may be rebuilt after switching versions; reading progress and bookmarks are stored
separately.

## Known limitations

- Thai dictionary breaking can be ambiguous for names, new vocabulary, and technical compounds.
- Font fallback may not visually match the selected Latin SD font.
- Markdown support is a reader-oriented subset, not complete CommonMark.
- Physical X3/X4 validation is still required before promoting this build from pre-release.
